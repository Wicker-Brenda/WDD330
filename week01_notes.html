Week 1 Notes

Doing Stuff With Web Things

  • Best Design Flow: Think, Design, Test, only then create code.
  • When creating code: html first, then javascript in logical order, testing as you go. CSS only after javascript is functional.
  • Use value to get user's input- document.getElementById("element_id").value
  • Use a section and innerHTML to show something to the user- document.getElementById("element_id").innerHTML
  • localStorage object: Allows you to save key/value pairs in a browser with no expiration date, read-only. The return value is a Storage object.
  • Syntax to SAVE data to localStorage: localStorage.setItem("key", "value");
  • Syntax to READ data from localStorage: var lastname = localStorage.getItem("key");
  • Syntax to REMOVE data from localStorage: localStorage.removeItem("key");
